# 列車位置ビューワ
既存のOudiaのエクスポートcsvデータを参照し、列車の在線状態を描画します。
列車がどこにいるのかわかるので、待ち時間がしんどくならないかもしれません。

## おことわり
このツールは各鉄道会社と何の関係もないので、このツールに関して鉄道会社さんに問い合わせたりしないでください。
時刻データは数年前のものなので、現在と時刻が違います。

## 起動
`App.java`のmain関数を実行してください。
